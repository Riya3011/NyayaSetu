Platform Features

Everything you need to
navigate the law

From document analysis to real-time consultation — NyayaSetu covers the full journey of legal assistance.

🤖

AI Legal Query Classifier

Automatically detects the legal category, sub-category, and urgency level from a user's description — and recommends the right type of lawyer.

📄

Document Summarization

Upload complex legal documents and receive concise, plain-language summaries highlighting key clauses, rights, obligations, and potential risks.

💬

Real-Time Secure Chat

Instant consultation between users and verified lawyers via a Socket.io-powered secure chat interface — no delays, no middlemen.

🔒

Role-Based Access Control

JWT-secured authentication with distinct roles for Users, Lawyers, and Admins — ensuring every user sees only what they should.

🎯

Intelligent Lawyer Matching

Queries are intelligently routed to lawyers based on their specialization, availability, and the assessed urgency of the issue.

⭐

Feedback & Rating System

Users can rate consultations and lawyers can build their reputation — maintaining quality, transparency, and accountability.

☁️

Encrypted Document Storage

All uploaded documents are stored securely via Cloudinary / AWS S3 with encryption, ensuring complete data privacy and controlled access.

📊

Admin Analytics Dashboard

Real-time platform analytics for administrators — manage users, verify lawyers, track query trends, and monitor overall system health.

AI Core Modules

Intelligence at the
heart of justice

Two powerful AI modules make NyayaSetu more than a directory — they make it a smart legal co-pilot.

🧠

Query Classification Engine

NLP-based module built on Python / FastAPI. Analyzes free-text legal queries and classifies them into category, sub-category, urgency (Low / Medium / High), and recommended lawyer type.

📑

Document Summarizer (LLM-based)

Processes uploaded PDFs and legal contracts using a Large Language Model. Extracts key clauses, flags risky terms, and generates a human-readable plain-language summary.

User Input
"My landlord is refusing to return my security deposit after I vacated the property 45 days ago and is not responding to messages."
▼ AI Classification Engine ▼
Classification Result
Category: Tenant / Rental Law Urgency: High Lawyer: Property Lawyer
Sub-category: Security Deposit Dispute · Legal action possible under Rent Control Act. Recommend immediate consultation.
Role-Based Access

Three distinct roles,
one unified platform

Each user type gets a dedicated, purpose-built experience tailored to their responsibilities.

👤

User / Citizen

Individuals seeking legal help

  • Submit legal queries in plain language
  • Upload documents for AI summarization
  • View matched lawyer recommendations
  • Real-time chat consultation
  • Rate and review lawyers
  • Track query history & status
⚖️

Lawyer

Verified legal professionals

  • Receive matched queries by specialization
  • View AI classification & urgency level
  • Access user-uploaded documents securely
  • Consult via real-time chat interface
  • Manage availability & case load
  • Build reputation through ratings
🛡️

Administrator

Platform management & oversight

  • Verify and approve lawyer registrations
  • Manage all user accounts
  • Monitor platform analytics & metrics
  • Resolve disputes and complaints
  • Configure AI classification rules
  • Oversee data security & access logs
